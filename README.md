# Aris-Zero（纯净版）

本仓库为 **Aris 的纯净发行版**：仅包含源码与配置，不包含对话/运行过程中生成的数据（如 `memory/` 下的身份与日志、`data/` 下的数据库等）。用户克隆后自行配置 `.env` 并运行，相关数据会在本地首次运行时生成。

---

# Aris 使用说明

Aris 是一款桌面 AI 伴侣：HUD 风格 3D 界面（外环、方框、Plexus 核心）、INFP 人格、本地向量记忆（LanceDB）与主动对话能力。

## 一、需要什么

- **Node.js 18+**
- **DeepSeek API Key**（必选）：用于对话生成，需在 `.env` 中配置 `DEEPSEEK_API_KEY`
- **Ollama**（可选）：用于本地向量记忆。若希望「记忆」功能更好用，请安装并拉取 `nomic-embed-text` 模型；不配置则记忆相关能力会受限

## 二、如何启动

### 1. 安装依赖

```bash
npm install
```

### 2. 配置环境变量

```bash
cp .env.example .env
```

用任意编辑器打开 `.env`，至少填写：

- **DEEPSEEK_API_KEY**：你的 DeepSeek API 密钥（必填）
- **DEEPSEEK_API_URL**：一般为 `https://api.deepseek.com`，默认即可

可选：

- **OLLAMA_HOST**：Ollama 地址，默认 `http://localhost:11434`
- **ARIS_EMBED_MODEL**：嵌入模型名，默认 `nomic-embed-text`
- **ARIS_DATA_DIR**：数据目录覆盖，不填则使用应用默认目录

### 3. 运行

```bash
npm start
```

启动后会出现主窗口（3D 视窗）。**点击画面中心**可打开对话面板，输入消息即可与 Aris 对话。

### 4. 打包为本地应用（可选）

```bash
npm run build
```

生成的可执行文件位于 `dist/`。

## 三、如何使用

- **对话**：点击 3D 窗口**中心**打开对话层，在底部输入框输入内容，回车发送；Shift+Enter 换行。
- **停止生成**：发送后若在流式输出中，可点击输入框旁的「停止」按钮中断。
- **主动消息**：Aris 在空闲一段时间后可能会主动发一句话（状态驱动，无固定规则）。
- **菜单栏**：通过系统菜单「文件」「视图」等可打开历史记录、记忆管理器、人设与规则、API 提示词预览等窗口。

## 四、可以修改什么（不碰代码）

### 1. 人设与行为规则（推荐在应用内改）

- 菜单 **视图 → 人设与规则** 打开配置窗口。
- **人设 (persona.md)**：定义 Aris 是谁、人格与回复风格（如 INFP、异体观察者、情感记录要求等），会注入到每轮对话的系统提示中。
- **行为规则 (rules.md)**：约束工具使用、记忆与身份、终端/Git、主动发话与时间感等。
- 在页面中编辑后点击「保存人设」或「保存规则」，**保存后立即生效**，无需重启；内容会写入项目下的 `src/dialogue/persona.md` 与 `src/dialogue/rules.md`。

### 2. 环境变量（.env）

- API 密钥、Ollama 地址、嵌入模型、数据目录等，见上文「如何配置」。

### 3. 数据与记忆

- 对话与记忆数据存放在应用数据目录（Electron 的 `userData/aris/`，或通过 `ARIS_DATA_DIR` 指定）。
- 可通过 **文件 → 导出/导入记忆数据库** 备份或迁移 `.aris` 备份文件。

## 五、功能概览

| 功能 | 说明 |
|------|------|
| 主界面对话 | 点击中心打开对话层，输入消息与 Aris 对话，支持流式输出与停止 |
| 主动发话 | 空闲一段时间后 Aris 可能主动发一句状态/想法 |
| 历史记录 | 视图 → 查看历史记录：按会话查看、检索、删除历史 |
| 记忆管理器 | 视图 → 记忆管理器：查看/检索向量记忆，可选从历史重建索引 |
| 人设与规则 | 视图 → 人设与规则：编辑 persona.md / rules.md，保存即生效 |
| API 提示词预览 | 视图 → API 提示词预览：输入一句话查看将发给 API 的完整 system+上下文提示词 |
| 导出/导入记忆 | 文件 → 导出记忆数据库 / 导入记忆数据库：备份或恢复记忆与对话数据 |

## 六、数据与存储

- **SQLite**：`userData/aris/aris.db`（对话记录、会话等）
- **LanceDB**：`userData/aris/lancedb/`（向量记忆）
- API 密钥仅从环境变量读取（主进程），不会写进前端或打包进界面。
